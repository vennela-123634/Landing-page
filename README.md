Task-01: Responsive Landing Page
This project is a responsive landing page designed to showcase an interactive navigation menu.
Features
Interactive Navigation: The menu dynamically changes its style or color when the user scrolls or hovers over specific items.
Fixed Positioning: The navigation bar is designed with a fixed position, ensuring it remains visible across all pages of the site.
Tech Stack:
HTML: Used to structure the navigation menu.
CSS: Used for styling, layout, and fixed positioning.
JavaScript: Used to add interactivity, such as triggering style changes during scrolling or hovering.
